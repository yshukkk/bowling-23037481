# Plan: 거터 게임 (Gutter Game)

## 목표
모든 굴림(roll)이 0핀인 게임(20번의 거터볼)의 점수는 0점이다.

## 추가할 파일/테스트
- `game.py`: `Game` 클래스 (roll, score 최소 골격)
- `test_game.py`: `test_gutter_game_scores_zero`
  - `Game()` 생성 후 `roll(0)`을 20번 호출
  - `score()`가 `0`을 반환하는지 검증

## 구현 범위
- `roll(pins)`: 핀 개수를 기록만 한다
- `score()`: `sum(rolls)` 같은 단순 합산이 아니라, 처음부터 **프레임 단위로 순회**하며 각 프레임의 두 굴림을 더해 총점을 계산한다 (10프레임 반복, 프레임당 두 굴림 소비). 이번 증분에서는 프레임 합산 로직만 있으면 거터 게임(전부 0)이 통과한다.
- 스페어, 스트라이크, 10프레임 보너스 로직은 범위 밖 (다음 증분에서 프레임 순회 로직에 조건을 추가하는 방식으로 확장)
